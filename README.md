# Odin Shopping Cart 🛒

Simple shopping cart project from The Odin Project React course. / React 80% /

[Live Demo](https://5x-shopping-cart.netlify.app/)

## Features

- Browse products
- Add to cart with quantity
- View and edit your cart (change quantity, remove items)
- Live total price
- WARNING: Not suitable for mobile

## Tech

- React + Vite
- React Router
- Context API for cart state
- Just plain CSS

## How to run it

```bash
git clone https://github.com/WujecDamian/odin-shopping-cart.git
cd odin-shopping-cart
npm install
npm run dev
